- 용어
SDK : 예를 들어, 페이스북에 우리의 서비스가 접근해서 제어하려면 페이스북에서 제공하는 API를 사용해야 한다. 근데 우리가 직접 API에 접근해서
제어할 수 있지만 방법이 조금 어렵다. 그래서 페이스북과 같은 회사들이 개발자들이 좀 더 쉽게 작업할 수 있게 하기 위해 우리가 이용할 수 있는 핸들
들을 만들어 준다. 그 핸들이 바로 SDK(software developer kit)라 한다.
쉽게 말해서 sdk를 이용해서 api들을 손쉽게 이용할 수 있다.

- 로그인 시스템을 구축하기 위해 필요한 것들
1. SDK를 로드한다.(우리의 웹페이지에)
2. SDK init(sdk를 초기화)(client ID와 같이 우리가 누군지 알려주어야한다.)
3. Login
4. Logout
5. isLogined(우리가 로그인 되어있는지 확인)
6. API 호출(여기서는 Facebook)
